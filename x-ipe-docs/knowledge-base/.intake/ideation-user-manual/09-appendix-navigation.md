### Navigation Elements

| Element | Location | Description |
|---------|----------|-------------|
| **Workspace Sidebar** | Far left | Top-level navigation: Ideation, Workflow, Settings |
| **Ideas Sidebar** | Left panel (within Ideation) | Folder tree with search, pin, collapse controls |
| **Content Area** | Center/right | Displays file content in view or edit mode |
| **Console** | Bottom panel (toggleable) | AI agent interaction area |
| **Header** | Top | Logo, navigation tabs, settings gear icon |

### Interactive Elements

**Sidebar Controls:**
- **рҹ”Қ Search box** вҖ” Filter folders/files by name (type to filter, results update live)
- **рҹ“Ғ Create folder** вҖ” Creates a new empty folder
- **рҹ“Ӯ Collapse all** вҖ” Collapses all expanded folder nodes
- **рҹ“Ң Pin** вҖ” Locks the sidebar open (by default it auto-hides)

**Folder Node:**
- **Click** вҶ’ Expand/collapse folder
- **Action buttons** (visible on hover):
  - **Add to** вҖ” Upload/add files to this folder
  - **Rename** вҖ” Rename the folder
  - **Enter View** вҖ” Navigate into the folder

**File Node:**
- **Click** вҶ’ Open file in content area
- **Action buttons** (visible on hover):
  - **в¬ҮпёҸ Download** вҖ” Download to local machine
  - **вңҸпёҸ Rename** вҖ” Rename the file
  - **рҹ—‘пёҸ Delete** вҖ” Delete the file

**Content Area Actions:**
- **рҹ“Ӣ Copy URL** вҖ” Copy a shareable link to the clipboard
- **вңҸпёҸ Edit** вҖ” Switch to edit mode
- **рҹӨ– Copilot Actions** вҖ” Open AI-powered actions menu
- **рҹ’ҫ Save** (edit mode) вҖ” Save changes
- **Cancel** (edit mode) вҖ” Discard changes

### Create Idea Modal

The Create Idea modal has three tabs:

| Tab | Icon | Purpose |
|-----|------|---------|
| **Compose Idea** | рҹ“қ | Write idea in Markdown editor |
| **Upload Files** | рҹ“Ғ | Drag-and-drop file upload |
| **UIUX Reference** | рҹҺЁ | Capture web page design elements |

Each tab has its own Submit/Create button at the bottom.
