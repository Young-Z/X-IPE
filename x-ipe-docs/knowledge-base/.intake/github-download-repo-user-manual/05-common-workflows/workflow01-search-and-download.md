# Workflow 01: Search and Download a Repository

## Goal

Find a specific repository on GitHub and download it to your local machine.

## Prerequisites

- Web browser open
- Git installed (for clone method) OR no tools needed (for ZIP method)

## Steps

1. Open your browser and navigate to **https://github.com**
   - **Expected outcome:** GitHub homepage loads with search bar visible at top

2. Click the **search bar** at the top of the page
   - **Expected outcome:** Search input is focused; you can start typing

3. Type the repository name (e.g., `x-ipe`) and press **Enter**
   - **Expected outcome:** Search results page shows matching repositories with descriptions and star counts

4. Click on the **repository name** (e.g., "Young-Z/X-IPE") in the results list
   - **Expected outcome:** Repository main page loads showing file tree, README, and action buttons

5. Click the green **"<> Code"** button near the top-right
   - **Expected outcome:** Dropdown opens showing Clone (HTTPS/CLI) and Download ZIP options

6. Click **"Download ZIP"** at the bottom of the dropdown
   - **Expected outcome:** Browser starts downloading `X-IPE-main.zip`; download progress shows in browser bar

7. Wait for download to complete, then extract the ZIP file
   - **Expected outcome:** Extracted folder contains the repository's source files

## Expected Result

You have a local copy of the repository's source code in your Downloads folder, ready to browse or use.
