# 1. Overview

## What is GitHub?

GitHub is the world's largest web-based platform for version control and collaboration using Git. It allows developers to host, review, and manage code repositories. GitHub provides a web interface for browsing repositories, managing issues, reviewing pull requests, and downloading source code.

## What Does "Downloading a Repository" Mean?

Downloading a repository means obtaining a local copy of a project's source code from GitHub to your computer. This is one of the most fundamental operations when working with GitHub. There are multiple methods to download a repository, each suited to different use cases:

1. **Clone with HTTPS** — Creates a full Git repository on your local machine with complete version history, enabling future pulls and pushes.
2. **Clone with GitHub CLI** — Uses the `gh` command-line tool for a streamlined clone experience with built-in authentication.
3. **Download ZIP** — Downloads a snapshot of the repository as a compressed ZIP file without Git history.
4. **Open with GitHub Desktop** — Opens the repository directly in the GitHub Desktop GUI application.
