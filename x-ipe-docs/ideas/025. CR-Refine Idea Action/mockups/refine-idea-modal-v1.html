Idea Summary

> Idea ID: IDEA-025

> Folder: 025. CR-Refine Idea Action

> Version: v1

> Status: Refined

Overview

A Change Request to implement the "Refine Idea" workflow action end-to-end — from the user clicking the action button in the workflow UI, through agent-driven refinement via CLI in an integrated terminal session, to automatic workflow status updates and deliverable display.

Key Features

  • Refine Idea Modal — readonly instructions + extra input + Copilot button
  • Console Session Management — find/create available terminal session
  • Skill Updates — accept extra_instructions, output to refined-idea/
  • Workflow Manager MCP — call update_workflow_action in DoD
  • Deliverable Viewer — inline markdown preview of refined output

Success Criteria

  • ☐ Modal opens with readonly instructions from copilot-prompt.json
  • ☐ Agent CLI command typed into console session
  • ☐ Refined output saved to {idea_folder}/refined-idea/
  • ☐ Workflow JSON updated via MCP
