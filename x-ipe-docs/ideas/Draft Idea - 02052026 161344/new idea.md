# 贪吃蛇 Web App (Snake Game)

## Idea Summary
Create a classic Snake game as a web application with modern UI and responsive design.

## Core Features
1. **Game Canvas** - A grid-based game area where the snake moves
2. **Snake Movement** - Control the snake using arrow keys or WASD
3. **Food System** - Random food appears on the canvas for the snake to eat
4. **Score Tracking** - Display current score and high score (saved to localStorage)
5. **Game States** - Start, Pause, Game Over screens
6. **Speed Increase** - Snake speeds up as score increases

## Technical Requirements
- Pure HTML5 Canvas + JavaScript (no external game libraries)
- Responsive design for mobile and desktop
- Modern gradient/glow visual effects
- Mobile touch controls (on-screen buttons)

## User Experience
- Simple one-click start
- Keyboard controls (Arrow keys / WASD)
- Visual feedback on eating food
- Game over with restart option

## Target Users
- Casual gamers
- People who want a quick break
- Nostalgia seekers who remember the classic Nokia Snake game
