# [FSN/WFJ/FBP] SMART POS Product Master & Price Master Integration Design

> Integration design for Product Master and Price Master data flow from AX to SMART POS for all three divisions, handling product and pricing reference data distribution.

## Integration Overview

Product Master and Price Master data flows from AX (ERP) to SMART POS via file-based integration through Apollo APAC and Apollo CN to Blob Storage.

### Key Facts
- **Divisions:** All three (FSN, WFJ, FBP)
- **Integration Type:** File
- **Gateways:** Apollo APAC, Apollo CN
- **Source:** AX (ERP)
- **Format:** SMART POS uses AX file format (no conversion needed)

## All Flows

| Division | Flow | AS-IS | To-Be | Type | Source | Target |
|----------|------|-------|-------|------|--------|--------|
| 3 Div | Product Master | AX → Share Folder → TTPOS | AX → new Share Folder → Apollo APAC → Apollo CN → Blob → SMART POS | File | AX | SMART POS |
| 3 Div | Price Master | AX → Share Folder → TTPOS | AX → new Share Folder → Apollo APAC → Apollo CN → Blob → SMART POS | File | AX | SMART POS |

**Notes:**
- Product Master: Each division has its own file folder
- Price Master: FBP uses one folder; FSN/WFJ share one folder

## Integration Findings & Assumptions

| Category | Finding/Assumption |
|----------|-------------------|
| Enterprise Direction | SMART POS uses AX file format — no conversion needed by Apollo |
| Integration Gateway | Apollo CN & Apollo APAC required |

## Low-Level Design — Flow Deep Dives

1. **Product Master** — AX → Apollo APAC → Apollo CN → Blob Storage → SMART POS (File)
2. **Price Master** — AX → Apollo APAC → Apollo CN → Blob Storage → SMART POS (File)
