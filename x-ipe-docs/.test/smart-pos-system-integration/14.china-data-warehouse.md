# [FSN/WFJ] SMART POS <-> China Data Warehouse Integration Design

> Integration design for SMART POS and China Data Warehouse (CDW) for FSN/WFJ divisions, handling multiple file-based data syncs including TX, VIP, Staff, Store, and item master data.

## Integration Overview

China Data Warehouse receives various data feeds from SMART POS. During parallel run, both TTPOS and SMART POS send data. The integration is file-based through Azure Blob Storage and Apollo CN to shared folders.

### Key Facts
- **Divisions:** FSN, WFJ
- **Integration Type:** File-based (bidirectional)
- **Data Flow:** SMART POS → Blob Storage → Apollo CN → Share Folder → CDW
- **Gateway:** Apollo CN
- **Parallel Run:** Both TTPOS and SMART POS send data concurrently
