# [WFJ] SMART POS <-> PEARL Integration Design

> Integration design for SMART POS and PEARL (product information system) for WFJ division, syncing product localization information including Chinese names, images, and fabric details.

## Integration Overview

PEARL provides product localization data to SMART POS for WFJ stores. SMART POS queries PEARL APIs to retrieve Chinese product names, images, fabric and color designations.

### Key Facts
- **Division:** WFJ only
- **Integration Type:** API
- **Domain:** Product
- **Direction:** SMART POS → PEARL (request/response)
- **Auth:** ADFS Clasp

## Integration Table View

| Division | Flow | Type | Request System | Response System |
|----------|------|------|----------------|-----------------|
| WFJ | Product Localization Information Sync | API | SMART POS | PEARL |

## PEARL API Interfaces

PEARL provides these APIs consumed by SMART POS:
- **Product list / detail**
- **SKU list / detail**
- **Referential list / detail**

Data retrieved by SMART POS:
- Product Image
- Product Chinese name
- Fabric designation in Chinese
- Color designation in Chinese

## Integration Findings & Assumptions

| Category | Finding/Assumption |
|----------|-------------------|
| Target System | PEARL provides Product, SKU, and Referential list/detail APIs |
| Integration Gateway | Apollo CN required for SMART POS & PEARL |
| Security | ADFS Clasp for authorization |
