# [FSN] SMART POS <-> China Product Hub Integration Design

> Integration design for SMART POS and China Product Hub for FSN division, providing localized product data (Chinese names, images, fabric/color designations).

## Integration Overview

China Product Hub provides enhanced product localization data for SMART POS, including product images, Chinese names, fabric designations, and other localized product information.

### Key Facts
- **Division:** FSN
- **Integration Type:** API
- **Gateway:** Apollo CN
- **Security:** ADFS Clasp for integration authorization
- **Performance:** Per API < 1.5 sec

## Business Flow Impact

| Domain | Function | Type | Request | Response |
|--------|----------|------|---------|----------|
| Product | Product Localization Information Sync | API | SMART POS | CN Product Hub |

## Integration Findings & Assumptions

| Category | Finding/Assumption |
|----------|-------------------|
| Target System | CN Product Hub provides two new API interfaces |
| Target System | API 1: Query all updated SPU Codes based on update time |
| Target System | API 2: Batch query product details by SPU codes |
| Integration Gateway | Apollo CN required for SMART POS & CN Product Hub |
| Security | ADFS Clasp used for integration authorization |

### Product Data Retrieved
- Product Image
- Product Chinese name
- Fabric designation in Chinese
- Color designation in Chinese
- Category for ICON
- Product short code
