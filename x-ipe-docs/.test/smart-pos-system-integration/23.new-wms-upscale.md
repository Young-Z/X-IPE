# [FSN/WFJ/FBP] SMART POS <-> New WMS (UpScale) Integration Design

> Integration design for SMART POS and New WMS (UpScale) for all three divisions. Similar to WMS but with updated infrastructure — syncs shipment orders for store receiving.

## Integration Overview

New WMS (UpScale) replaces the legacy WMS with an upgraded system. Shipment order sync to SMART POS follows a similar pattern but with updated flow paths.

### Key Facts
- **Divisions:** All three (FSN, WFJ, FBP)
- **Integration Type:** File
- **Gateway:** Apollo CN
- **Domain:** Inventory
- **Key difference from WMS:** Apollo needs to do filtering (unlike legacy WMS)
