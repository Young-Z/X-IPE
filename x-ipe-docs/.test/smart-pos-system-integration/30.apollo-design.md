# SMART POS -> Apollo Design

> Apollo CN integration gateway configuration and domain name reference for SMART POS integration. This is a reference/hub page linking to Apollo-related design documents.

## Integration Overview

Apollo CN is the central integration gateway for all SMART POS integrations. This page documents the Apollo CN domain names across environments and links to related design documents.

### Key Facts
- **Role:** Central integration gateway for all SMART POS integrations
- **Components:** Apollo CN (China), Apollo APAC (distribution)
- **Environments:** DEV, SIT, UAT, OAT, PRD
