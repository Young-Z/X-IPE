# [FSN/WFJ] SMART POS <-> TTPOS VIP/Staff VIP Quota Integration Design

> Integration design for SMART POS providing VIP and Staff VIP quota check services to TTPOS during parallel run, within the Sales Management domain.

## Integration Overview

During parallel run, TTPOS calls SMART POS to check VIP and Staff VIP quota calculations. SMART POS provides the quota computation logic that TTPOS consumes via API.

### Key Facts
- **Divisions:** FSN, WFJ
- **Integration Type:** API
- **Domain:** Sales Management
- **Direction:** TTPOS → SMART POS (request/response)
- **Auth:** ADFS Clasp for integration authorization

## Integration Table View

| Division | Flow | Type | Request System | Response System |
|----------|------|------|----------------|-----------------|
| FSN/WFJ | VIP/Staff Quota Check | API | TTPOS | SMART POS |
