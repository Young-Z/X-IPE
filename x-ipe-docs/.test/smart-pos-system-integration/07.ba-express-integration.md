# [FBP] SMART POS <-> BA Express Integration Design

> Integration design for SMART POS and BA Express for FBP division, covering real-time and daily TX data sync and sales target reporting.

## Integration Overview

BA Express is a reporting/analytics system. SMART POS pushes sales transaction data and sales target data to BA Express for FBP division stores.

### Key Facts
- **Division:** FBP only
- **Integration Types:** API (realtime), File (daily batch)
- **Gateway:** Apollo CN
- **Parallel Run:** TTPOS sends TTPOS TX data while SMART POS sends SMART POS TX data (different source system codes)

## Business Flow Impact

| Domain | Function | Type | Source | Target | Downstream |
|--------|----------|------|--------|--------|------------|
| Sales Management | Sync Realtime TX Data | API | SMART POS | BA Express | Report |
| Sales Management | Sync Daily TX Data | File | SMART POS | BA Express | Reconciliation with realtime TX data |
| Sales Management | Sync Daily SaleTarget Data | File | SMART POS | BA Express | Report |

## Integration Findings & Assumptions

| Category | Finding/Assumption |
|----------|-------------------|
| Enterprise Direction | During parallel run, TTPOS and SMART POS both send TX data |
| Enterprise Direction | SMART POS TX data uses different source system code from TTPOS |
| Integration Gateway | Apollo CN required |
| Security | Defined at flow level |

## Low-Level Design — Flow Deep Dives

1. **Sync Realtime TX Data** — SMART POS → BA Express (API)
2. **Sync Daily TX Data** — SMART POS → BA Express (File)
3. **Sync Daily SaleTarget Data** — SMART POS → BA Express (File)
