# [FSN/WFJ/FBP] SMART POS <-> LION Integration Design

> Integration design for SMART POS and LION (CRM system) for all three divisions. LION is one of the most complex integrations with multiple bidirectional flows including TX, CX, VIP, deposits, and sample management.

## Integration Overview

LION is the CRM system that handles customer experience (CX) data, VIP management, and related functions. It consists of LION CDM (Customer Data Management) and LION RMC (Relationship Management Center). This is one of the richest integration designs with many flow types.

### Key Facts
- **Divisions:** All three (FSN, WFJ, FBP)
- **Integration Types:** API and File (mixed)
- **Gateways:** Apollo CN
- **LION subsystems:** LION CDM, LION RMC, LION OSS

## All Flows Between LION & SMART POS

| Division | Flow | Type | Source | Target | Notes |
|----------|------|------|--------|--------|-------|
| 3 Div | TX Data | API | SMART POS | LION CDM | Same format, same source system code |
| 3 Div | CX Query | API | SMART POS | LION RMC | Via Apollo CN |
| 3 Div | CX One Id Query | API | SMART POS | LION CDM | New (no AS-IS equivalent) |
| FBP | CX Registration | API | SMART POS | LION RMC | |
| FBP | VIP Searchlog | File | SMART POS | LION CDM | Parallel: merge tools combine TTPOS/SMART POS data |
| FBP | Sample Redeem | File | LION CDM | TTPOS/SMART POS | Apollo routes based on store white list |
| FSN/WFJ | Deposit Header & Details | File | SMART POS | LION CDM | Parallel: combine tools merge both sources |
| FSN/WFJ | Store Master | File | SMART POS | LION CDM | Keep AS-IS in parallel |
| FSN/WFJ | Product Master | File | SMART POS | LION CDM | Keep AS-IS in parallel |
| FSN/WFJ | Staff Master | File | SMART POS | LION CDM | Keep AS-IS in parallel |
| FSN/WFJ | Item Category | File | SMART POS | LION CDM | Keep AS-IS in parallel |

## Low-Level Design — Flow Deep Dives

1. **Sync Realtime TX Data** — SMART POS → Apollo CN → LION CDM (API)
2. **Sample Redeem** — LION → OSS → Apollo CN → Blob Storage → SMART POS (File)
3. **VIP Searchlog** — SMART POS → Blob Storage → LION OSS → LION (File)
4. **TX Additional Data** — additional TX-related data flow
