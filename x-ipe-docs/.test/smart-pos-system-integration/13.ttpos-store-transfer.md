# [FSN/WFJ] SMART POS <-> TTPOS Store Transfer Integration Design

> Integration design for inter-store transfers between SMART POS and TTPOS during the parallel run phase, handling TS (Transfer Shipment) and TSAC (Transfer Shipment Acknowledgement) file exchanges.

## Integration Overview

During the migration period, some stores run SMART POS while others still run TTPOS. This integration handles inter-store transfers between the two POS systems using file-based exchanges (TS and TSAC files).

### Key Facts
- **Divisions:** FSN, WFJ
- **Integration Type:** File-based (TS/TSAC files via Azure Blob Storage)
- **Gateway:** Integration Layer
- **Domain:** Inventory / Store Transfer

## Transfer Approaches

### Approach 1: SMART POS BTQ → TTPOS BTQ
1. SMART POS initiates transfer to TTPOS BTQ
2. Integration Layer sends TS file to TTPOS
3. TTPOS handles TS file
4. When BTQ receives transfer in, TTPOS sends TIAC (ACK) to AX
5. TTPOS sends TSAC file back to SMART POS via Integration Layer

### Approach 2: TTPOS BTQ → SMART POS BTQ
1. TTPOS initiates transfer to SMART POS BTQ
2. Integration Layer sends TS file to SMART POS
3. When BTQ receives transfer in, SMART POS sends TIAC (ACK) to Integration Layer
4. SMART POS sends TSAC file to TTPOS via Integration Layer

## Technical Implementation

### New Components
- Program for SMART POS → TTPOS TS (insert Transfer Out records into XF_TransQueue)
- Program for SMART POS → TTPOS TSAC (insert Transfer In records into XF_TransQueue)
- Program for TTPOS → SMART POS Transfer Out (generate TS file)
- Program for TTPOS → SMART POS Transfer In ACK (generate TSAC file)

### New Database Tables
- **XF_SM_WHITE_STORE:** White store list (stores switched to SMART POS)
- **XF_SM_TRANSEXPORT:** TS/TSAC file export task table

### Modifications
- Modify AS-IS Trigger `XF_Trans.xf_trans_ax_ins`
- Data Server application processes XF_TransQueue entries

### Technical Decisions
- Shared folder uses Azure Blob Storage ✓
- Program deployable on SMART POS VM (can connect to TTPOS T2 Database) ✓
- Development in Java ✓
