## Description

Add the `skill-meta-x-ipe-knowledge.md` template to the skill-creator templates. This was explicitly deferred in the original FEATURE-059-A specification (Out of Scope, line 222: "Skill-meta template for knowledge type — if needed, add in a follow-up"). All other 6 active skill types have corresponding skill-meta templates; knowledge is the only one without.

## Business Justification

Consistency across the skill type system. Without a skill-meta template for knowledge skills, agents creating knowledge skills lack acceptance criteria guidance, leading to inconsistent quality. The knowledge skill type has unique characteristics (operations with phase backbone, writes_to discipline, stateless service model) that require type-specific acceptance criteria not covered by other templates.
