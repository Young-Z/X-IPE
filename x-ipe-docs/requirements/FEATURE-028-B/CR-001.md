## Description

Replace the single bilingual `copilot-instructions.md` template (with `---LANG:en---` / `---LANG:zh---` markers) with two separate files:
- `copilot-instructions-en.md`
- `copilot-instructions-zh.md`

When user chooses a language via `--lang`, copy the corresponding file directly instead of extracting sections from a single file.

## Justification

- Simpler to maintain: each file is self-contained, no markers needed
- Easier to review/edit: contributors work with standard markdown
- Simpler scaffold logic: direct file copy instead of extraction
