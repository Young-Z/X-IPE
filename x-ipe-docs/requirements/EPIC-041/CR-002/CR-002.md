# CR-002: Action Context System

> Epic: EPIC-041
> Source: IDEA-029 (CR-Optimize Feature Implementation Part 2)
> Status: Proposed
> Created: 2026-02-26

## Summary

Extends EPIC-041 with:
1. **Deliverable Tagging** — `$output:name` / `$output-folder:name` replaces `deliverable_category`
2. **Action Context** — template-driven input declarations per action
3. **Modal Context UI** — dropdowns from template schema with auto-detect + reopen support
4. **Instance Persistence** — keyed deliverables + context field
5. **Skill Extra Context** — `extra_context_reference` for explicit file paths

## Impact on Existing Features

| Feature | Impact |
|---------|--------|
| FEATURE-041-A | Extended — context resolution replaces `_resolveInputFiles()` |
| FEATURE-041-B | Superseded — `action_context` dropdowns replace `input_source` multi-dropdown |
| FEATURE-041-C | Extended — skills gain `extra_context_reference` |
| FEATURE-041-D | Extended — new actions need `action_context` definitions |
