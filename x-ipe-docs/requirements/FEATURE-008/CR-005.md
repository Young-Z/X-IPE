## Acceptance Criteria

## Technical Approach

### Copy URL Button
- Location: `workplace-editor-actions` div, between status and Edit button
- Icon: Bootstrap `bi-link-45deg` or `bi-clipboard`
- URL format: `{origin}/api/file/content?path={encodedPath}&raw=true`
- Use `navigator.clipboard.writeText()` for clipboard access

### File Rename (Verification Only)
- Already implemented in `workplace.js` lines 324-333 and 569-649
- Backend API exists at `POST /api/ideas/rename-file`
