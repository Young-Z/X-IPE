## Description

Add `x-ipe-tool-kb-librarian` tool skill creation to FEATURE-049-F scope. The skill handles the AI-powered organization of intake files: moving files to appropriate KB folders, assigning 2D tags (lifecycle × domain), and generating frontmatter metadata. Currently, FEATURE-049-F built the intake infrastructure (upload to `.intake/`, status tracking, UI, command trigger) but excluded the actual skill that processes the files.

## Business Justification

The requirement document (HR-049.6) explicitly mandates: *"Run AI Librarian button triggers agent CLI session via `x-ipe-tool-kb-librarian` skill"* and *"AI Librarian auto-handles: move files to appropriate folders, assign tags, generate frontmatter."* Without this skill, the "✨ Run AI Librarian" button sends a command that has no skill to receive it — the feature is incomplete.
